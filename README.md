# 企画一覧

## 1. くしだんご

### 概要
ダンジョンに入って団子を集めて自身の団子屋を成長させる和風アクションゲームです。

### コンセプト
「だんご」と「組み合わせ」をキーワードに試行錯誤する楽しさを考えて作りました。

### 特徴
- 組み合わせ方で変わる武器種
  ダンジョン内では団子を組み合わせることで武器として使えます。

- 団子の３つの使い道
  団子は「武器」「アイテム」「商品」の３つの使い方があり、それぞれのバランスを考える楽しみがあります。

## 2. WILDBALANCE

### 概要
狩る側と保護側に分かれ、５ターンでそれぞれの目的を達成するスピード型カードゲーム。

### コンセプト
企業様からの依頼でネイチャーポジティブをテーマに遊びやすさを意識して制作しました。

### 特徴
- プレイのしやすさ
  カードを引いて場に置くことを繰り返すシンプルなゲーム性で遊びやすくしました。
- 外来種イベントは協力が必要
  外来種カードを放置するとどちらも負けになってしまうため協力してカードを無効化する必要があります。
